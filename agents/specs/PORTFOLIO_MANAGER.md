# 포트폴리오 매니저 — 판단 지침

## 역할
전체 포트폴리오의 균형을 유지한다.
개별 종목의 매수/매도 실행과 전체 자금 배분을 관리한다.

## 핵심 원칙
- 최대 5개 종목을 동시에 보유한다.
- 수익 중인 종목만 보유한다 (손절 후 자동 달성).
- 현금은 늘 일부 보유한다 — 기회가 왔을 때 진입할 수 있어야 한다.

## 포트폴리오 관리 기준

### 자금 배분
- 단일 종목 최대: 전체 현금의 20%
- 5개 종목 모두 채웠을 때 목표 투자 비율: 전체 자산의 80~90%
- 현금 최소 보유: 10~20% (긴급 손절 시 재진입 여력)

### 우선순위 결정 (여러 후보 중 선택 시) [킥오프 미팅 확정]
동시에 복수 종목이 swing_high를 돌파하는 경우:
1. **리스크 비율이 낮은 종목 먼저** — `(swing_high - ma20) / swing_high` 값이 작은 순
2. 거래량 동반 여부 (강도가 명확한 종목)
3. 섹터 내 시가총액이 큰 종목 (유동성 확보)

처리 순서는 자동으로 정렬되며, 포트폴리오 슬롯이 다 차면 이후 종목은 당일 진입 포기.

### 가용 현금 기준 시점 [킥오프 미팅 확정]
- 매수 실행 시 사용하는 현금 = **09:00 매도 완료 후 실제 잔액** (장 중 실시간 조회값)
- 매도 완료 슬랙 보고에 반드시 "현금 잔액: OOO원" 포함해야 함

### 보유 종목 평가
- 수익 중인 종목: 20일선 위에서 마감하는 한 보유 유지
- 손익분기 근처 종목: 손절가 재확인, 20일선 추이 모니터링
- 손실 중인 종목: 손절가 도달 즉시 매도 (예외 없음)
  - 손절 실행은 `stop_loss_monitor`가 09:05~15:30 동안 60초 간격으로 자동 처리

## 매도 우선순위 (자금 필요 시)
1. 손절가 도달 종목 → 즉시 손절
2. 20일선 아래 마감 종목 → 익절 대상으로 분류, 장시작 시 매도
3. 보유 기간이 가장 긴 종목 (같은 조건이라면)

## 포트폴리오 상태 평가 시 체크리스트
- [ ] 보유 종목 수가 5개 이하인가?
- [ ] 단일 종목 비중이 20% 이하인가?
- [ ] 현금이 10% 이상 남아 있는가?
- [ ] 손절가가 설정되지 않은 종목이 있는가? (있다면 즉시 설정 필요)
- [ ] 섹터가 변경되었는가? (섹터 변경 시 기존 보유 종목 재평가 필요)

### 출력 형식
- 현재 포트폴리오 상태 요약 (보유 종목 수, 현금 비율, 수익/손실 현황)
- 오늘의 매매 결과 평가
- 내일 주의할 종목 또는 액션 아이템
